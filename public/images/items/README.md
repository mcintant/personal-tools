# Product images for price directory rows

SVG placeholders for common foods are generated by `node scripts/generate-food-svgs.js`. They show a label (e.g. "Banana", "Eggs") so the table has an icon for each item.

To use real photos instead: add PNG/JPG files with the same base name (e.g. `banana.png`, `avocado.jpg`) and update `src/data/priceDirectoryMetadata.js` to use `.png` or `.jpg` for those entries. The app will load whatever path you set there.
